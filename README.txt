# Night at the Museum
Starter project for the Udacity [VR Developer Nanodegree](http://udacity.com/vr) program.


### Versions Used
- [Unity LTS Release 2017.4.4](https://unity3d.com/unity/qa/lts-releases?version=2017.4)
- [GVR SDK for Unity v1.100.1](https://github.com/googlevr/gvr-unity-sdk/releases/tag/v1.100.1)

 ### Submitting for Android

 ### There are two scenes in the project: "Lobby", "Museum".
To get to the actual project scene - "Museum", the "ENTER" button should be pressed in the first scene "Lobby", then the "Museum of Education" button on the faculty panel should be pressed.
